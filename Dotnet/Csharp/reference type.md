- Variables of reference types store references to their data instead of the data themselves
- Keywords used to create reference types: 
	- [[class]]
	- [[interface]]
	- [[Delegates|delegate]]
	- [[record]]
- Built-in reference types:
	- [[dynamic]]
	- object
	- [[string]]
