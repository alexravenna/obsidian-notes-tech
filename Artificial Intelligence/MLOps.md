# Description
- Inspired by [[DevOps]]
- A methodology for increasing efficiency and automation for:
	- Model reproducibility
	- Model validation
	- Model deployment
	- Model retraining
- Model registry: like a [[Repository|code repository]] for ML models; for saving and versioning
- Save training datasets centrally
