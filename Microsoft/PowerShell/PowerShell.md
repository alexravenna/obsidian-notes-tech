# cmdlets
- A compiled command
- Can be developed in [[.NET]] or [[NET Core|.NET Core]]
- Named according to a "verb-noun" standard
	- `Get-Verb` returns all standard verbs
- `Get-Command` lists all available cmdlets
	- can be filtered with `-Noun` and/or `-Verb`
