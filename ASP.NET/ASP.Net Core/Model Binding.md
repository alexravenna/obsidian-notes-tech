- Defines how to map inputs from [[HTTP request]]s to model properties
- Inputs can be from:
	- [[form fields]]
	- [[request body]]
	- [[route]] data
	- [[query string parameters]]
	- uploaded files
- [[Model validation]] occurs after binding
