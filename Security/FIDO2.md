# Description
- An open standard managed by the [[FIDO Alliance]] for [[passwordless]] [[authentication]]
- Incorporates the [[WebAuthn]] standard
- FIDO2 supports:
	- hardware-based [[authentication]]
	- web browser supports
	- 1. registration
	- 2. authentication
		- challenge sent: message encrypted with user's public key
	- 3. verification
