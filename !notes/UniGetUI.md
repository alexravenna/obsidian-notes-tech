# Description
- Formerly called WingetUI
- A [[desktop]] [[application]] for multiple [[package manager]]s on [[Windows]]
- Supports:
	- [[WinGet]]
	- [[Scoop]]
	- [[Chocolatey]]
	- [[pip]]
	- [[npm]]
	- [[.NET]]
	- [[PowerShell Gallery]]
	- [[cargo]]
	- [[vcpkg]]
