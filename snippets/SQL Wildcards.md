| Wildcard | Description                                     |
| -------- | ----------------------------------------------- |
| %        | Zero or more characters                         |
| _        | A single character                              |
| []       | Any single character within the brackets        |
| ^        | Any character not in the brackets               |
| -        | Any single character within the specified range |
