# Description
- Built on [[Azure App Service]]
- Use the [[WebJobs SDK]]
- Enables running a [[program]] or [[script]] in the same context as an [[app]]
- Types of programs supported:
	- [[.exe]]
	- [[Java]]
	- [[PHP]]
	- [[[Python]]
	- [[Node.js]]
- Types of scripts supported:
	- [[.cmd]]
	- [[.bat]]
	- [[PowerShell]]
	- [[Bash]]
- Can be:
	- scheduled
	- run by trigger
- Often used for background jobs
